About us

ພວກເຮົາເປັນນັກສຶກສາ 5 ຄົນ ທີ່ຮ່ວມມືກັນໃນການພັດທະນາເວັບໄຊທ໌ເພື່ອນຳສະເຫນີຕໍ່ຄູອາຈານໃນວິຊາທີ່ກ່ຽວຂ້ອງກັບການພັດທະນາເວັບ. ພວກເຮົາໄດ້ເລືອກໃຊ້ເທັກໂນໂລຢີໃນການພັດທະນາເວັບໃຫ້ມີຄວາມໃຊ້ງານງ່າຍ, ຄວາມສະຖຽນ, ແລະສາມາດປັບແຕ່ງໄດ້ຕາມຄວາມຕ້ອງການ. ໂຄງການຂອງພວກເຮົາປະກອບໄປດ້ວຍການອອກແບບ UI/UX ໃຫ້ສວຍງາມ ແລະໃຊ້ງານໄດ້ສະດວກ, ການປັບແຕ່ງເວັບໃຫ້ຮອງຮັບການໃຊ້ງານບນອຸປະກອນຕ່າງໆ, ແລະການນຳໃຊ້ HTML, CSS ມາໃຊ້ໃນການຄວບຄຸມການເຄື່ອນໄຫວຂອງເວັບໄຊທ໌. ພວກເຮົາຍັງໃຊ້ Git, GitHub ເເລະ GitHub-Desktop ສຳລັບການຄວບຄຸມເວີຊັນ ແລະການຮ່ວມມືຂອງທີມ, ເຊິ່ງເຮັດໃຫ້ການທຳງານເປັນທີມມີຄວາມສະດວກຫຼາຍຂຶ້ນ. ໃນຂະນະການພັດທະນາໂຄງການນີ້, ພວກເຮົາໄດ້ຮຽນຮູ້ຫຼາຍຢ່າງ ເຊັ່ນການຂຽນໂປຣແກຣມຢ່າງມີໂຄດສະອາດ, ການຈັດການໂປຣເຈັກ, ແລະການເເກ້ໄຂບັນຫາທີ່ເກີດຂື້ນລະຫວ່າງທຳງານ. ນອກຈາກນີ້, ການທຳວຽກເປັນທີມຍັງຊ່ວຍໃຫ້ພວກເຮົາພັດທະນາຄວາມສາມາດໃນການສື່ສານ ແລະການເຮັດວຽກຮ່ວມກັນຢ່າງມີປະສິດທິພາບ.
